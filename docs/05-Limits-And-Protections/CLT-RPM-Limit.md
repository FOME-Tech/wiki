# Coolant Based RPM limit

The CLT based RPM limit is a useful limit to put in place to prevent the car from being over-driven whilst it is cold. It can be found under __Base Engine > Limits and fallbacks__ and can be enabled by turning on the __CLT based RPM curve__.
